# Zenn AI News Project Rules

## 📋 プロジェクト概要
海外のAI関連ニュースを自動的に収集・要約し、Zennに投稿するシステム

---

## 🎯 記事選定ルール

### 基本方針
- 幅広い読者層を対象とした記事作成
- AI関連の最新アップデート・トレンド・ニュースを中心に扱う
- 技術的に深すぎず、一般のエンジニア・AI興味層が理解できる内容
- 注目度が高く、話題性のある記事を優先的に選定

### ターゲット読者
- エンジニア（AI専門家でなくてもOK）
- AI技術に興味がある一般層
- ビジネスでAI活用を検討している人
- 最新AIトレンドを追いかけたい人

### 選定基準

#### 1. コンテンツタイプ（優先度順）
1. 最新アップデート - GPT-5リリース、新モデル発表等
2. トレンド - 話題のAI技術・手法・アプリケーション
3. ニュース - AI業界の動き、企業発表、規制関連
4. 実用事例 - AIの実際の活用例、成功事例

#### 2. ソース優先度
Tier 1（最優先）:
- TechCrunch
- The Verge
- Ars Technica
- VentureBeat AI
- AI公式ブログ（OpenAI, Google AI, Anthropic等）

Tier 2（推奨）:
- MIT Technology Review
- Wired
- Hacker News（高スコア記事）
- Reddit r/MachineLearning（人気投稿）

Tier 3（補足）:
- Medium（AI著名人の記事）
- 企業技術ブログ
- AI系YouTuberの話題

#### 3. 注目度指標
- ソーシャルメディアでのシェア数（100+推奨）
- Hacker Newsスコア（100ポイント以上推奨）
- Reddit upvotes（500+推奨）
- コメント数・エンゲージメント
- 記事の新鮮度（7日以内を優先）

#### 4. 記事の分かりやすさ基準
採用基準:
- 専門用語の適切な説明がある
- 具体例・ユースケースが示されている
- 図表・画像で視覚的に理解しやすい
- 一般エンジニアが読んで理解できるレベル

除外基準:
- 数式だらけの学術論文
- 過度に専門的な実装詳細のみ
- 広告・宣伝目的の記事
- 信頼性の低い情報源
- 既に日本語で広く報道済みの話題

## 記事作成ルール

### 人間らしさを重視した記事執筆方針

**最重要原則: AIが書いた記事っぽくしない**

- 機械的な列挙ではなく、会話的なトーン
- 個人的な視点・感想を適度に含める
- 完璧すぎる文章を避け、自然な言い回しを使う
- 絵文字は使用禁止
- 定型文・テンプレート的表現を避ける

### 人間らしい文章の特徴
- 「〜だと思います」「〜かもしれません」など、適度な曖昧さ
- 「個人的には」「興味深いのは」など、書き手の視点
- 「ただし」「一方で」など、自然な接続詞
- 箇条書きばかりでなく、段落で説明
- 完璧な整理ではなく、話の流れを重視

### Zenn記事フォーマット
```markdown
---
title: "[記事タイトル]"
emoji: "📝"
type: "tech"
topics: ["AI", "機械学習", "ニュース"]
published: false
---

## 元記事
[原題] - [ソース名]
[URL]

## どんな内容？

[元記事の要約を会話的に説明。3-4段落程度]

## 注目すべきポイント

[箇条書きではなく、段落形式で重要な点を説明]

## 技術的な補足

[必要に応じて技術的な解説を追加]

## 所感

[個人的な感想・考察を含める]
```

### 記事作成時の必須ルール
1. 必ず `published: false` で作成
2. 元記事のURL・タイトルを明記
3. 著作権・引用ルールを遵守
4. 機械翻訳そのままではなく、自然な日本語に
5. 技術用語は文脈に応じて解説
6. 絵文字は使用禁止（emojiフィールドのみOK）

### 避けるべき表現
- 「〜について説明します」系の導入
- 「以下のポイントが挙げられます」
- 「〜することができます」の多用
- 完璧に整理された箇条書きのみの構成
- 感情のない機械的な要約

## 🛠️ 技術仕様

### 使用ツール
- **Zenn CLI**: 記事管理
- **GitHub**: バージョン管理・自動デプロイ
- **Claude API**: 要約生成
- **WebSearch/WebFetch**: ニュース収集

### ファイル構成
```
zenn-ai-news-project/
├── articles/           # Zenn記事
├── scripts/           # 自動化スクリプト（作成予定）
│   ├── collect_news.py
│   ├── summarize.py
│   └── publish.py
├── config/            # 設定ファイル（作成予定）
│   └── sources.json
└── ZENN_AI_NEWS_RULES.md  # 本ファイル
```
